# StockPilot 실행 · 확인 가이드

지금까지 만든 것(백엔드 + 프론트엔드)을 처음부터 켜서 확인하는 순서입니다.
터미널 3개를 쓴다고 생각하면 편해요: ① 백엔드 ② 프론트 ③ 테스트/명령.

---

## 0. 사전 준비 (최초 1회)

1. **`.env` 확인** (StockPilot 루트) — 아래 키가 채워져 있어야 실데이터·LLM·로그인이 동작합니다.
   ```
   UPSTAGE_API_KEY=...       # Solar (응답 생성)
   NAVER_CLIENT_ID=...       # 뉴스
   NAVER_CLIENT_SECRET=...
   DART_API_KEY=...          # 공시
   SUPABASE_URL=...          # RAG·즐겨찾기·로그인 저장
   SUPABASE_KEY=...
   KRX_ID=...                # 시세 재무지표(선택)
   KRX_PW=...
   JWT_SECRET=아무_긴_랜덤문자열   # 로그인 토큰 서명(없으면 개발용 기본값)
   ```
2. **Supabase 스키마 적용** — Supabase 대시보드 → SQL Editor에서 `data/supabase_schema.sql` 전체 실행.
   (documents·watchlists·glossary_terms·**users** 테이블 생성. 이미 실행했으면 재실행해도 안전 — `if not exists`)
3. **용어 사전 적재(선택, 용어사전 기능용)**
   ```
   uv run python data/scripts/ingest_rag.py
   ```

## 1. 백엔드 실행 (터미널 ①)

StockPilot 루트에서:
```
uv sync
uv run uvicorn app.main:app --reload
```
- 로그에 `StockPilot API 시작` / `✅ StockPilot 그래프 컴파일 완료` 가 뜨면 정상.
- 주소: http://127.0.0.1:8000  (문서: http://127.0.0.1:8000/docs)

### 백엔드 단독 확인 (/docs)
브라우저에서 http://127.0.0.1:8000/docs 열고 "Try it out":

| 확인 | 방법 | 기대 결과 |
|---|---|---|
| 헬스체크 | `GET /api/v1/health/` | `{"status":"healthy",...}` |
| 채팅(단건) | `POST /api/v1/chat/` body `{"message":"삼성전자 요즘 어때?","session_id":"t"}` | Solar가 쓴 등락률·원인 분석 문장 |
| 회원가입 | `POST /api/v1/auth/register` `{"username":"test","password":"1234"}` | `access_token` 반환 |
| 로그인 | `POST /api/v1/auth/login` 같은 값 | `access_token` 반환 |
| 즐겨찾기 | 우측 상단 **Authorize**에 토큰 넣고 `POST /api/v1/watchlist/` `{"ticker":"삼성전자"}` → `GET /api/v1/watchlist/` | 저장·목록 확인 |

## 2. 프론트엔드 실행 (터미널 ②)

`frontend` 폴더에서:
```
cd frontend
npm install          # 최초 1회 (ogl, motion 등 설치)
npm run dev
```
- 주소: http://localhost:5173

### 화면 확인 체크리스트
- [ ] 배경에 **emerald 오로라(Aurora)** 가 흐르고, 스크롤해도 아래가 검게 끊기지 않음
- [ ] 상단 **반투명 내비바** — 종목 분석 / 급등·급락 / 용어 사전 / 로그인
- [ ] **시작하기** 클릭 → 채팅창이 부드럽게 등장, 왼쪽에 **내 대화** 목록(LineSidebar) 표시
- [ ] 종목(예: 삼성전자) 입력 → **큰 등락률(상승 빨강/하락 파랑) + 현재가 + 원인 분석(스트리밍) + 출처 버튼**
- [ ] 다른 종목(예: SK하이닉스) 다시 질문 → 색·화살표 정상, 왼쪽 목록에 대화 추가
- [ ] 후속 질문("왜 올랐어?") → 직전 종목으로 이어서 답변
- [ ] 내비 **용어 사전** → PER 설명, **급등·급락** → 상승 이슈 종목 목록
- [ ] **로그인 → 회원가입 탭** → 아이디·비번·비번확인 단계별 입력(빈 값이면 다음 안 넘어감) → 가입 완료 시 자동 로그인
- [ ] 왼쪽 목록 항목 클릭 → 그 대화 복원 / **🗑 현재 대화 삭제** 동작
- [ ] **새로고침** → 대화 목록 유지되고, 대화가 무한으로 늘어나지 않음

## 3. 테스트 (터미널 ③)

StockPilot 루트에서:
```
uv run pytest -q
```
- `passed` 로 끝나면 정상. (경고 1개는 테스트 도구 관련이라 무시 가능)
- 핵심 흐름만 보고 싶으면:
  ```
  uv run pytest tests/test_chat_flow.py tests/test_auth.py -v
  ```

## 자주 겪는 문제

- **루트(`/`)에서 404** → 정상. 엔드포인트는 `/api/v1/...` 아래, 문서는 `/docs`.
- **PowerShell에서 `curl` 오류** → `curl` 대신 `curl.exe` 사용.
- **로그인/즐겨찾기가 500** → Supabase에 `users` 테이블 SQL(0단계 2번) 미적용.
- **응답이 딱딱한 템플릿(불릿)으로만 나옴** → `UPSTAGE_API_KEY` 미설정(Solar 미연동 폴백).
- **프론트에서 "서버에 연결하지 못했어요"** → 백엔드(uvicorn)가 안 켜져 있음.
